# Expense Tracker (CLI)

A simple backend-focused command-line expense tracker built with Python.  
This project is part of the Backend Roadmap practice.

## Features
- Add, update, and delete expenses
- View all expenses
- View total expenses
- View monthly summary
- Data persistence using JSON

## Requirements
- Python 3.8 or higher

## How to Run
git clone https://github.com/yourusername/expense-tracker.git  
cd expense-tracker  
python expense_tracker.py  

Follow the on-screen menu to manage expenses.

## Notes
- Expenses are saved in `expenses.json` in the same folder.
- Pure backend logic: no GUI, just Python CLI.
- Focuses on CRUD operations, input handling, and data persistence.
